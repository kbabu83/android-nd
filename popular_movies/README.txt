Note about the TMDB API key
======================================
TMDB API's require an API key for access. Please update yours in a source file APIKey.java in the package com.infinity.popularmovies. Template shown below.


APIKey.java
======================================

package com.infinity.popularmovies;

public interface APIKey {
    public static final String TMDB_API_KEY = <your_api_key_here>;

}
